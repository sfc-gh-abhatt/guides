## How to Use

Each guide is self-contained in its own directory with a `README.md` explaining prerequisites, execution order, and sample interactions. Clone the repo and follow the instructions:

```bash
git clone https://github.com/sfc-gh-abhatt/guides.git
cd guides/<guide-name>
# Follow the README.md inside each guide
```

## Built with Cortex Code

All content in this repository was generated using **Cortex Code** — Snowflake's AI-powered development environment.

### Get Cortex Code

| Platform | How to Access |
|----------|---------------|
| **Desktop (macOS)** | Download from [Snowflake Cortex Code](https://docs.snowflake.com/en/user-guide/cortex-code/cortex-code) |
| **Snowsight** | Available in-browser under AI & ML in your Snowflake account |
| **CLI** | Install via `snow cortex` commands in [SnowCLI](https://docs.snowflake.com/en/developer-guide/snowflake-cli/index) |

### What Cortex Code Can Do

- Generate SQL, Python, and dbt projects grounded in your Snowflake data
- Build Cortex Agents, Semantic Views, and Search Services interactively
- Explore your account's tables, views, and schemas with natural language
- Create Streamlit apps, notebooks, and data pipelines
- Debug, refactor, and optimize existing code
